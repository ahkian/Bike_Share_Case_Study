# Bike Share Case Study
This is the code portion of a capstone project for the Google Data Analytics Course. 
## Goal of the Analysis
With the datasets given find differences in bicycle usage between yearly subscribers and casua; users (i.e. those who pay per a usage) and propose a new marketing strategy to get more casual users to subscribe. 
### Three Main Questions
1. How do annual members and casual riders use Cyclistic bikes differently?
2. Why would casual riders buy Cyclistic annual memberships? 
3. How can Cyclistic use digital media to influence casual riders to become members?
